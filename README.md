# Dex-contract

This repository contains the smart contract code for a decentralized exchange (DEX) developed by SupaDao.

## Overview

This project implements a decentralized exchange (DEX) on the Monad blockchain. The DEX allows users to trade various ERC20 tokens in a trustless and non-custodial manner. This is achieved through smart contracts deployed on the Monad mainnet (or testnets).

## Features

- **Decentralized Trading:** Users can swap ERC20 tokens directly with each other using the DEX smart contract.
- **Liquidity Provision:** Liquidity providers can add liquidity to different trading pairs, earning fees generated by trades.
- **ERC20 Compliance:** The smart contract adheres to the ERC20 token standard for compatibility with a wide range of tokens.
- **Governance:** Future versions may incorporate governance mechanisms to allow token holders to vote on protocol changes.
